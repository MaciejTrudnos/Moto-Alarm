# MotoAlarm #

Projekt alarmu do motocykla wysyłający wiadomości sms w momencie wykrycia zmiany położenia, zasilany powerbankiem

### Użyte podzespoły ###

* Arduino Nano
* Przetwornica LM2596
* Moduł SIM800L
* Kondensator 1000uf
* 3-Osiowy Akcelerometr i Żyroskop I2C MPU-6050
