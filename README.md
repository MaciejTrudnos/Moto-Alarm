# MotoAlarm #

Motorcycle alarm design\
After change position it make phone call\
Managed by SMS commands and powered by a powerbank

## Components

* Arduino Nano
* Boost Converter LM2596
* SIM800L module
* Capacitor 1000uf
* 3-Axis Accelerometer and Gyroscope I2C MPU-6050
